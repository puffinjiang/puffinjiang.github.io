记录一下开启Ubuntu服务器的ssh远程连接过程

SSH服务安装#

1. 检查服务是否开启#

通过下面命令检查服务是否开启

1

ps -e | grep sshd

2. 安装服务#

1

apt install openssh-server

3. 启用服务#

1

systemctl enable ssh

4. 启动服务#

1

systemctl start sshd.service

服务配置#

配置文件路径#

注意： 使用的是sshd而不是ssh，ssh是ssh-client相关的，sshd是ssh-server相关信息，要注意区分！

1

vim /etc/ssh/sshd_config
