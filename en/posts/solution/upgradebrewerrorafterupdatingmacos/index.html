背景#

今天在我更新完苹果推送的系统更新之后，打开vscode时，弹窗提示 Git无法使用 之类的信息。让我以为更新之后Git被删除了， 于是我使用 brew upgrade git 命令想看一下之前安装过的Git是否还在

解决方法#

运行 xcode-select --install ，此命令会安装会帮助我们安装 Command Line Tools，安装完成后即可正常使用 brew 命令了。
